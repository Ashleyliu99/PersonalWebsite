Seattle Real Estate Analytics

An end-to-end SQL project built on a simulated Greater Seattle real estate dataset. This project demonstrates relational database design, a Python-based data ingestion pipeline, and business-driven SQL queries that surface pricing trends, market speed, and buyer/seller behavior across 9 cities and 3 property types.

Step 2

Data Pipeline

A Python script reads 6 CSV files and bulk-inserts all rows into MySQL. Tables are loaded in foreign key dependency order to avoid constraint violations.

Python  ·  insert_data.py (core loop)
# Load in FK order: users → agents → sellers → buyers → houses → transactions
CSV_TABLES = ["users", "agents", "sellers", "buyers", "houses", "transactions"]

for table in CSV_TABLES:
    df = pd.read_csv(os.path.join(DATA_PATH, f"{table}.csv"))
    df = df.where(pd.notnull(df), None)       # convert NaN → SQL NULL

    cols         = ", ".join(df.columns)
    placeholders = ", ".join(["%s"] * len(df.columns))
    sql          = f"INSERT INTO {table} ({cols}) VALUES ({placeholders})"

    cursor.executemany(sql, [tuple(row) for row in df.values])
    conn.commit()
    print(f"✅  {len(df):,} rows inserted → {table}")
Key design choices:  Parameterized %s placeholders prevent SQL injection · Column names are read directly from CSV headers (no hardcoding) · pd.notnull() converts Pandas NaN to Python None, which MySQL stores as NULL · executemany() inserts all rows in a single round-trip per table.

Step 3

Business Analysis

SQL queries written to answer real business questions — using GROUP BY, JOIN, aggregate functions, and subqueries to surface market intelligence.

Q1

Seller Motivation Breakdown

Why are sellers entering the market? Understanding motivation informs pricing strategy and negotiation approach.

SQL
SELECT
    selling_motivation,
    COUNT(*) AS total_sellers,
    ROUND(
      COUNT(*) * 100.0
      / (SELECT COUNT(*) FROM sellers),
      1
    ) AS percentage
FROM sellers
GROUP BY selling_motivation
ORDER BY total_sellers DESC;
  • Relocation is the #1 driver (36%) — job moves fuel supply
  • Downsizing + estate sales = 37% — aging homeowner cohort
  • Investment exits at 8.7% suggest some cooling of investor activity

Q2

Listing vs. Sale Price by City & Property Type

How close are final sale prices to listing prices? A ratio above 100% means buyers bid over asking.

SQL
SELECT
    h.city,
    h.property_type,
    COUNT(*)                   AS total_sold,
    FORMAT(AVG(h.listing_price), 0)
                               AS avg_list_price,
    FORMAT(AVG(t.sale_price), 0)
                               AS avg_sale_price,
    ROUND(
      AVG(t.sale_price / h.listing_price * 100),
      1
    )                          AS sale_to_list_ratio
FROM transactions t
JOIN houses h ON t.house_id = h.house_id
GROUP BY h.city, h.property_type
ORDER BY sale_to_list_ratio DESC;
  • Condos in Bellevue fetch 102.4% of asking — highest competition
  • Issaquah single-family homes sell at 102.1% — strong demand suburb
  • Sammamish townhouses at 98.1% are the only segment below list price
